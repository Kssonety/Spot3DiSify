# Spot3DiSify
**FOR THIS TO WORK, YOU NEED A NINTENDO 3DS OR 2DS RUNNING CUSTOM FIRMWARE.**

**THIS PROGRAM WILL RUN BEST ON A NEW NINTENDO 3DS XL.**

**THIS PROGRAM WAS TESTED ON A NEW NINTENDO 2DS XL.**

Spotify for the Nintendo 3DS Family!

## Compilation

1. Install devkitPro from https://devkitpro.org

2. Run `git clone https://github.com/Kssonety/Spot3DiSify`

3. Run `cd Spot3DiSify`

4. Run `make`

## Instructions

1. Download the latest release in either .3dsx or .cia format, and place it on your 3ds's SD card.

2. Create the SD/3ds/Spot3DiSify/music folder if you haven't already and place your .ogg files inside.

3. If you got a .CIA file, use FBI to install it onto your homescreen.

4. If you got a .3DSX file, open Homebrew Launcher and run the 3dsx file from there.
